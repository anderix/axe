Card Grid

The .grid class creates a responsive auto-fill layout. Cards can be <article> elements (static content) or <a> tags (clickable cards). Both get surface background, border, and hover treatment.

Details / Disclosure

What is Axe?

Axe is a lightweight, semantic-first CSS framework. Write plain HTML and it looks good. No classes required for basic elements.

How do I use it?

Create a brand.css with your colors, fonts, and shape preferences. Import it alongside axe.css in your project. Then write semantic HTML.

Can I customize it?

Everything visual is controlled by CSS custom properties in brand.css. The framework never hard-codes colors, fonts, or radii. Use the brand builder tool to generate your brand file.
